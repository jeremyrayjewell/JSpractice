Iterators

  • let's you iterate over a collection
  • implement your own iterators to define custom behavior
  • iterable vs. iterator

When is an object iterable? When it provides a means for you to retrieve an iterator for it.


Example:

const numbers = [1, 2, 3];
const numbersIterator = numbers[Symbol.iterator]();

console.log(numbersIterator.next());
console.log(numbersIterator.next());
console.log(numbersIterator.next());
console.log(numbersIterator.next());

for (const n of numbers){
console.log(n);
}
